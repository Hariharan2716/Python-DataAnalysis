## Sales Insight Analysis (Pandas)

### The Idea

- This is logic building and beginner project that helped me to understand pandas and how data analyzation works.
- This project analyzes a sales dataset using Python and Pandas to extract meaningful business insights.

### Tech Stack
- Python
- Pandas

### What was the Idea
- Data cleaning handling missing values, formatting dates and extracting months.
- Analysing the sales by product, region, and time.
- To sort the products and high-value orders
